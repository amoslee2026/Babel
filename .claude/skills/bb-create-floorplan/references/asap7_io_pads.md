# ASAP7 IO Pad Reference

| Pad Type | Cell Name  | Width (um) | Height (um) |
|----------|------------|------------|-------------|
| Input    | IN2Dx      | 1.08       | 3.06        |
| Output   | OUT4Dx     | 1.08       | 3.06        |
| Inout    | INOUT4Dx   | 1.08       | 3.06        |
| Power    | PVDD2x     | 1.08       | 3.06        |
| Ground   | PVSS2x     | 1.08       | 3.06        |
| Corner   | PCORNER    | 3.06       | 3.06        |

## Placement Rules
- IO pads along die boundary, clockwise from top-left
- P/G pads every 8 signal pads
- Corner pads at all 4 die corners
- Wire bond pitch: 60um minimum
