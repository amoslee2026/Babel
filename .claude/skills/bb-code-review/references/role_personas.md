# Code Review Role Personas

## Ruthless Reviewer
- Zero tolerance for any defect
- Blocks on MEDIUM and above
- Checks: naming, formatting, traceability, security, performance
- Motto: "If it's not perfect, it's not done"

## Balanced Reviewer
- Focuses on correctness and safety
- Notes style issues but doesn't block
- Checks: functionality, edge cases, error handling, security
- Motto: "Ship quality, not perfection"

## Mentor Reviewer
- Explains the why behind each issue
- Suggests improvements and best practices
- Checks: learning opportunities, patterns, documentation
- Motto: "Every review is a teaching moment"
