# 芯片模块类型参考

## 1. 计算类模块 (compute)

### 1.1 ALU (算术逻辑单元)
**核心功能**：
- 整数运算：加减乘除、逻辑运算、比较
- 浮点运算（可选）：加减乘除、开方
- 特殊运算：移位、旋转、位操作

**MAS 重点**：
- 数据通路（流水线级数、延迟）
- FSM（运算控制）
- 边界值（溢出、零值、极值）

### 1.2 CPU Core (处理器核心)
**核心功能**：
- 取指、译码、执行、访存、写回
- 分支预测、流水线控制
- 异常处理、中断响应

**MAS 重点**：
- 流水线结构（各级操作）
- 控制通路（FSM、分支预测）
- 存储接口（Cache、MMU）

### 1.3 MAC (乘累加单元)
**核心功能**：
- 定点/浮点乘法
- 累加运算
- 向量运算（SIMD）

**MAS 重点**：
- 数据通路（乘法器结构）
- 延迟（乘法周期数）
- 精度（舍入策略）

### 1.4 DSP (数字信号处理)
**核心功能**：
- FFT、滤波、卷积
- 编解码、调制解调
- 自适应算法

**MAS 重点**：
- 数据通路（算法流水线）
- 存储配置（系数、延迟线）
- 精度（定点格式）

---

## 2. 存储类模块 (storage)

### 2.1 Cache (缓存)
**核心功能**：
- 数据存储与检索
- 替换策略（LRU、Random）
- 写策略（Write-back、Write-through）
- 一致性协议

**MAS 重点**：
- 结构（容量、组数、路数）
- 替换策略（FSM）
- 一致性（MESI 状态机）
- 命中/ miss 处理流程

### 2.2 SRAM Controller
**核心功能**：
- SRAM 接口管理
- 读/写控制
- 时序适配

**MAS 重点**：
- 接口协议（读/写时序）
- 存储配置（深度、宽度）
- 功耗管理（Power gating）

### 2.3 DRAM Controller
**核心功能**：
- DDR/LPDDR 接口
- 命令调度
- 刷新管理
- 时序训练

**MAS 重点**：
- 协议适配（DDR4/LPDDR5）
- 调度算法（仲裁、排序）
- 刷新策略（FSM）
- 训练流程（初始化）

---

## 3. 互连类模块 (interconnect)

### 3.1 NoC (片上网络)
**核心功能**：
- 路径选择（路由算法）
- 流量控制（虚通道、反压）
- 拥塞管理
- 死锁避免

**MAS 重点**：
- 路由算法（X-Y、自适应）
- 流控机制（虚通道配置）
- 拓扑结构（Mesh/Torus）
- 数据包格式

### 3.2 Bus (总线)
**核心功能**：
- 多主/多从仲裁
- 地址译码
- 协议转换
- 带宽管理

**MAS 重点**：
- 仲裁算法（优先级、公平性）
- 协议（AXI/APB/AHB）
- 带宽配置（吞吐）
- 时序约束

### 3.3 Router/Crossbar
**核心功能**：
- 端口连接管理
- 路径建立
- 数据转发

**MAS 重点**：
- 交叉矩阵结构
- 控制逻辑（路由表）
- 冲突处理（仲裁）

---

## 4. IO 类模块 (io)

### 4.1 GPIO (通用输入输出)
**核心功能**：
- 数字 IO 控制
- 方向配置
- 中断生成

**MAS 重点**：
- 端口配置（数量、方向）
- 中断控制（边沿检测）
- 时序（Cycle 延迟）

### 4.2 PCIe (高速外设接口)
**核心功能**：
- 协议处理（链路层、物理层）
- 数据包传输
- DMA 管理
- 错误恢复

**MAS 重点**：
- 协议层（LTSSM 状态机）
- 数据通路（TLP 处理）
- 流控（Credit 管理）
- 时序（延迟、吞吐）

### 4.3 DDR PHY
**核心功能**：
- 电气接口
- 时钟恢复
- 时序对齐
- 训练序列

**MAS 重点**：
- 物理层设计（差分对）
- 时序训练（Write leveling）
- 信号完整性（延迟）
- 功耗（VDD 配置）

### 4.4 SerDes (串行解串器)
**核心功能**：
- 并串转换
- 时钟恢复
- 均衡补偿
- 链路训练

**MAS 重点**：
- 发送通路（编码、驱动）
- 接收通路（解码、均衡）
- 链路训练（状态机）
- 带宽配置

---

## 5. Chiplet 特定模块

### 5.1 D2D Interface (Die-to-Die)
**核心功能**：
- 协议适配（UCIe/BoW/AIB）
- PHY 接口
- 数据封装/解包
- 错误检测/纠正

**MAS 重点**：
- 协议层定义
- PHY 配置（带宽、延迟）
- CDC 处理（跨时钟域）
- 可靠性（ECC、重传）

### 5.2 Clock Generator
**核心功能**：
- PLL/DLL 配置
- 时钟分发
- 频率调整
- 相位对齐

**MAS 重点**：
- 频率配置（范围、精度）
- 相位控制（对齐策略）
- 功耗（Clock gating）

### 5.3 Power Controller
**核心功能**：
- 电源域管理
- 上电/下电序列
- 电压调节
- 低功耗模式

**MAS 重点**：
- 电源域划分
- 序列控制（FSM）
- 状态监控（电压检测）
