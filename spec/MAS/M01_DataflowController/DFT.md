# M01_DataflowController — DFT Specification

## 1. 扫描链配置

| 参数              | 值                          |
|-------------------|-----------------------------|
| 扫描链数量        | 4 条                        |
| 扫描触发器数量    | ~1200（估算，SF4 4nm）      |
| 扫描模式          | Full-scan，LSSD 兼容        |
| 扫描时钟          | scan_clk（独立，与 CLK_SYS 隔离） |
| 扫描使能          | scan_en（高有效）           |
| 故障覆盖率目标    | >= 95%（stuck-at fault）    |

### 扫描链分配

| 链 ID | 内容                              | 触发器数（估算） |
|-------|-----------------------------------|-----------------|
| SC0   | 主 FSM 状态寄存器 + 控制逻辑      | ~300            |
| SC1   | 线程上下文寄存器 CTX[0/1]         | ~300            |
| SC2   | 指令队列（OP_QUEUE）寄存器        | ~400            |
| SC3   | 性能计数器 + 中断逻辑             | ~200            |

### 扫描接口信号

| 信号名       | 方向 | 描述                    |
|--------------|------|-------------------------|
| scan_en      | IN   | 扫描使能，高有效        |
| scan_clk     | IN   | 扫描时钟                |
| scan_in[3:0] | IN   | 4 条链串行输入          |
| scan_out[3:0]| OUT  | 4 条链串行输出          |

## 2. 指令队列 MBIST

OP_QUEUE 使用 SRAM 宏（16×64b），需独立 MBIST 控制器。

| 参数              | 值                          |
|-------------------|-----------------------------|
| MBIST 算法        | March-C（覆盖 SA/TF/CF）    |
| MBIST 时钟        | mbist_clk（最高 250 MHz）   |
| 测试模式          | mbist_en 高有效，与正常模式互斥 |
| 修复支持          | 1 行冗余（单行修复）        |
| 测试完成信号      | mbist_done（脉冲）          |
| 测试结果信号      | mbist_pass（高=通过）       |

### MBIST 接口信号

| 信号名       | 方向 | 描述                    |
|--------------|------|-------------------------|
| mbist_en     | IN   | MBIST 使能              |
| mbist_clk    | IN   | MBIST 专用时钟          |
| mbist_done   | OUT  | 测试完成脉冲            |
| mbist_pass   | OUT  | 测试结果（1=通过）      |
| mbist_repair | OUT  | 修复使能（连接 eFuse）  |

### MBIST 与正常模式隔离

```
mbist_en=1 时：
  - OP_QUEUE SRAM 端口切换至 MBIST 控制器
  - m00_op_valid 强制拉低（防止误发指令）
  - AXI 主接口强制 arvalid=0
```

## 3. JTAG 接口

M01 通过顶层 JTAG TAP 访问，支持边界扫描和寄存器直读。

| 参数              | 值                          |
|-------------------|-----------------------------|
| JTAG 标准         | IEEE 1149.1-2013            |
| TAP 位置          | 芯片顶层统一 TAP，M01 为从节点 |
| IR 长度           | 5 bit                       |

### JTAG 指令集（M01 相关）

| IR 编码 | 指令名       | 功能                              |
|---------|--------------|-----------------------------------|
| 5'b00001 | BYPASS      | 标准 bypass                       |
| 5'b00010 | IDCODE      | 读取模块 ID（M01=32'hA001_0001）  |
| 5'b00100 | READ_STATUS | 读取 STATUS 寄存器                |
| 5'b00101 | READ_PERF   | 读取 PERF_CNT0/1/UTIL             |
| 5'b00110 | FORCE_RESET | 强制 CTRL[1]=1（软复位）          |
| 5'b01000 | SCAN_EN     | 进入扫描模式（配合 scan_en 信号） |

### JTAG 信号（顶层透传）

| 信号名 | 方向 | 描述        |
|--------|------|-------------|
| tck    | IN   | JTAG 时钟   |
| tms    | IN   | 模式选择    |
| tdi    | IN   | 数据输入    |
| tdo    | OUT  | 数据输出    |
| trst_n | IN   | 异步复位    |
