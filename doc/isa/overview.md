# ISA 总览

## 设计目标

针对 TinyStories 15M LLM 推理的专用处理器，覆盖以下算子：Embedding、MatMul、RMSNorm、RoPE、Attention、Softmax、SwiGLU、Residual、Sampling。

> **参考模型**：ISA 基于 stories260K (dim=64) 进行设计分析，向量宽度固定64元素，与目标模型算子需求一致。

**约束**：
- 向量宽度固定64元素（对应dim=64）
- FP32为主，INT8量化可选
- 无通用CPU指令，仅推理专用

---

## 寄存器文件

### 向量寄存器

| 名称 | 数量 | 宽度 | 说明 |
|------|------|------|------|
| `v0`–`v31` | 32个 | 64×FP32 = 2048 bit | 通用向量寄存器 |

约定：`v0` 硬连线为零向量；`v1`–`v3` 保留给 Q/K/V 临时缓冲。

### 标量寄存器

| 名称 | 数量 | 宽度 | 说明 |
|------|------|------|------|
| `s0`–`s15` | 16个 | 32 bit FP32 | 标量运算、循环计数、地址偏移 |

约定：`s0`=0.0（硬连线）；`s1`=1.0（硬连线）；`s14`=SP；`s15`=PC（只读）。

### 特殊寄存器

| 名称 | 宽度 | 说明 |
|------|------|------|
| `acc` | 64×FP32 | MatMul累加器（隐式） |
| `kv_ptr` | 32 bit | KV Cache当前写指针 |
| `head_id` | 4 bit | 当前注意力头编号（0–7） |
| `s_dim` | 16 bit | MMUL矩阵行数（由MSET_DIM设置） |
| `status` | 8 bit | 标志位（zero / overflow / done） |

## 内存模型

```
地址��间（32-bit）：

0x0000_0000  权重 ROM         4MB   只读  embedding table, W_q/k/v/o, W_ff1/2/3
0x0040_0000  激活 SRAM        256KB 读写  向量中间结果
0x0044_0000  KV Cache SRAM   512KB 读写  [head][seq][dim]
0x004C_0000  查找表 ROM       64KB  只读  exp / sin / cos / sigmoid / sqrt
```

访问粒度：向量 256字节对齐（64×FP32）；标量 4字节。

## 指令编码格式

固定32-bit编码，4种格式：

```
V型（向量三操作数）
31      26 25    21 20    16 15    11 10     6 5       0
┌─────────┬────────┬────────┬────────┬────────┬────────┐
│ OPCODE  │   VD   │  VS1   │  VS2   │  VS3   │  FUNC  │
│  6 bit  │  5 bit │  5 bit │  5 bit │  5 bit │  6 bit │
└─────────┴────────┴────────┴────────┴────────┴────────┘

VI型（向量+立即数）
31      26 25    21 20    16 15                        0
┌─────────┬────────┬────────┬───────────────────────────┐
│ OPCODE  │   VD   │  VS1   │          IMM16            │
│  6 bit  │  5 bit │  5 bit │          16 bit           │
└─────────┴────────┴────────┴───────────────────────────┘

M型（内存访问）
31      26 25    21 20    16 15    11 10                0
┌─────────┬────────┬────────┬────────┬──────────────────┐
│ OPCODE  │   VD   │  BASE  │   SD   │    OFFSET11      │
│  6 bit  │  5 bit │  5 bit │  5 bit │    11 bit        │
└─────────┴────────┴────────┴────────┴──────────────────┘

S型（标量/控制）
31      26 25    21 20                                  0
┌─────────┬────────┬──────────────────────────────────────┐
│ OPCODE  │   SD   │               IMM21                  │
│  6 bit  │  5 bit │               21 bit                 │
└─────────┴────────┴──────────────────────────────────────┘
```

## 指令分类汇总

| 类别 | 指令数 | OPCODE范围 | 覆盖算子 |
|------|--------|-----------|---------|
| 向量算术 | 6 | 0x00–0x05 | Residual, RMSNorm, SwiGLU, Attention |
| 矩阵乘法 | 3 | 0x08–0x0A | MatMul |
| 特殊函数 | 5 | 0x10–0x14 | RMSNorm, RoPE, Softmax, SwiGLU |
| 归约 | 4 | 0x18–0x1B | RMSNorm, Softmax, Sampling |
| 内存访问 | 6 | 0x20–0x25 | 所有算子 |
| KV Cache | 3 | 0x28–0x2A | Attention |
| 标量/控制 | 5 | 0x30–0x34 | 循环、跳转 |
| **合计** | **32** | | |
