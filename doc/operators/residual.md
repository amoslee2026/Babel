# Residual 算子

## 功能

残差连接 - 将子层输出加到原始输入，稳定梯度流。

---

## 公式

$$x_{out} = x_{in} + f(x_{in})$$

---

## 参数

| 参数 | 值 |
|------|-----|
| dim | 64 |
| 调用频率 | 10次/forward |

---

## 代码实现

```c
for (int i = 0; i < dim; i++) {
    x[i] += output[i];
}
```

---

## 计算复杂度

64次加法 = 64 FLOPs × 10 = 640 FLOPs/forward

---

## 硬件实现

最简单算子，64个加法器并行执行，1 cycle完成。
